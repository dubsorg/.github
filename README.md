# .github
This repo contains shared GitHub assets (for example, workflow templates) and team-shared Copilot customizations.

## Copilot custom agents

Org-shared custom agents live under `.github/agents/` in this repo.

To use an agent in another repository:
1. Copy the desired `*.agent.md` file(s) into that repository at `.github/agents/`.
2. In VS Code, open Copilot Chat and pick the agent by name (for example, **Spellcheck**).
